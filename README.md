# Sales Analysis

The analysis was done using Excel. The analysis was aimed at identifying what drives business growth using sales dataset aggregated over a period of 6 months. The analysis compared the performance of sales team based on the revenue generated for the company. Also included in the project is a month-on-month analysis of the revenue generated by the company. 
Recommendations were provided based on the insights discovered from the analysis. 

### The analysis project involved:
- ➡️ Data preparation 
- ➡️ Data cleaning 
- ➡️ Merging the different tables (orders, stores, sales teams, products) into one single table using vlookup and xlookup.
- ➡️ Analysis of the dataset using pivot tables
- ➡️ Dashboard visualization
- ➡️ Insights and Recommendations
